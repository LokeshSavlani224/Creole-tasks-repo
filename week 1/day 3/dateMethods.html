getFullYear()is used to Get year as a four digit number (yyyy)
getMonth() is used to Get month as a number (0-11)
getDate() is used to Get day as a number (1-31)
getDay() is used to Get weekday as a number (0-6)
getHours() is used to Get hour (0-23)
getMinutes() is used to Get minute (0-59)
getSeconds() is used to Get second (0-59)
getMilliseconds() is used to Get millisecond (0-999)
getTime() is used to Get time (milliseconds since January 1, 1970)
